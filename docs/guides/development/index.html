Development

The Specviz monorepo contains the core library and documentation website, as well as many individual example starter application packages.

.

├── bun.lockb
├── examples
│   ├── example-advanced-annotation
│   ├── example-basic-annotation

│   ├── example-basic-spectrogram
│   └── example-interactive-spectrogram
├── package.json
├── packages
│   ├── specviz
│   └── website
│       └── static
├── readme.md

Runtime and Dependencies​

After installing the bun JavaScript runtime and package manager, install the development dependencies.

$ bun install
bun install v1.1.29 (6d43b366)

1116 packages installed [3.77s]

Blocked 3 postinstalls. Run `bun pm untrusted` for details.

Examples​

To run a starter project, change to the example's directory and run the dev script.

$ cd examples/example-advanced-annotation
$ bun dev

The example webserver will display the connection URL. In the default case, http://localhost:5173.

VITE v5.4.8  ready in 205 ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
➜  press h to show help
